**Content fields:**

- Mothers name (Bride side)

### [HIGH] Sortable & Toggle Sections

All sections on the invitation page must be **sortable** (drag to reorder) with **enable/disable** and **background image upload** support per section.

| Section | Sortable | Can Toggle Off | Background Image Upload |
|---|---|---|---|
| Landing | Yes — **always pinned at top** | **No — always visible** | Yes |
| Undangan/Ucapan | Yes | Yes | Yes |
| Event Details | Yes | No — always visible | Yes |
| RSVP | Yes | No — always visible | Yes |
| Photos Carousel | Yes | Yes | No |
| Wishes | Yes | Yes | Yes |

**Rules:**
- Landing section is always the first section — cannot be reordered or disabled
- Disabled sections are hidden from the guest invitation page but remain in the builder
- Background image upload replaces the section background; builder should show a preview
- Sort order and toggle state are persisted per event

2. **Starting-point templates** — pre-designed but user can continue customizing after selecting (colors, text, images, section order, etc.)

Both types should cover the full invitation page including all sections.
